# Helpful shortcuts to learn:

## Cursor navigation within a line
- `b` - goes to the start of the previous
- `w` - goes to the start of the next word

## Edit
- `I` to insert at the beginning of a line

## State modifiers
- `u` - undo previous change
- `ctrl` + `r` redo a change that was undone

## How to switch between buffers?
- `<leader>,` is a very nice shortcut to be honest, it opens up telescope and then you can do whatever in it.

## How to paste?
- `p` paste after cursor
- `gp` paster cursor and move cursor after the pasted text.
